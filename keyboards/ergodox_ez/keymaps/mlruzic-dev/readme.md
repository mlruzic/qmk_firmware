# ErgoDox EZ keymap

## Differences with default layout is in the symbol layer
 - removed media layer (moving a mouse with a keyboard has never been useful to me)
 - symbol layer: all parens available in the home row (I use these a lot for programming)
 - added layer with some git macros I use often
 - mapped arrows to home row (right) for convenience
